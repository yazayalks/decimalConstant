Описание программы

Программа DecimalConstant является текстовым редактором с наличием анализатора текста десятичной константы. Это значит, что данная программа предназначена для работы с текстом по выявлению ошибок в записи десятичной константы. В ней мы можем вводить текст и редактировать его. Давайте запустим DecimalConstant. Для этого щелкаем мышкой по значку программы. Откроется окно программы.

Описании функций

Создать

Создает новый документ.

Открыть

Открывает документ с расширением *.rtf

Сохранить

Созраняет документ с расширением *.rtf

Созранить как

Созраняет документ с измененным названием и расширением *.rtf

Выход

Закрывает программу

Отменить

Отменяет последнее совершенное действие

Повторить

Возвращает последнюю совершенную отмену

Вырезать

Вырезает выделеный текст в буфер обмена

Копировать

Копирует текст в буфер обмена

Вставить

Вставить текст из буфера обмена

Удалить

Удалить выделеный текст

Выделить всё

Выделяет весь текст в текстовом поле.
